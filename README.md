# Codelab
An easy to use API to compile, run and evaluate code.  

To compile the program:  
  In the src folder, run "javac *.java && java Main"
  
To test the API:  
  In the tests folder, run the following commands  
  chmod +x vpl_compile.sh vpl_run.sh  
  ./vpl_compile.sh  
  ./vpl_execution  
  ./vpl_run.sh  
  ./vpl_execution
